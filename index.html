About Me

As a sophomore at Lehigh University, I study Computer Science and Business. I've been programming since eighth grade, with most of my time spent doing full-stack web development. I started programming in C# for competitions in my freshman year of high school, and I've been building servers, administrating Linux and Windows systems, and building web applications since 2014.

Most recently, I've been using ASP.NET Core to build web applications in C# with Vue.js as a reactive frontend framework. I have configured and maintained LEMP stacks, Plesk servers, Windows Servers, WordPress installations, and custom SQL databases. I worked at Application X from May of 2016 to January of 2020 as a full stack developer.

Lehigh Valley Live - Easton Area School District App
November, 2018

The Easton Area School District alert app was developed by teacher Michael Tanczos and alumnus Connor Greene. It consolidates multiple streams of district information including district-issued alerts, social media posts and links to lunch menus, the online lunch payment site and archived school board meetings on YouTube.
"The app is going to send you messages. It's going to give you information. It's going to build a stronger community within the Easton Area School District," according to Lisa Manzo.

The Morning Call - NHS Attendance Program
January, 2018

National Honor Society members at Easton Area High School might be good students, but until recently, their meetings never started on time. It would take up to 20 minutes for officers to take attendance for the nearly 260-member group. It was an ongoing problem that school employees and students hadn't been able to solve for years. Until now.
Check-in time has been cut in half or more thanks to an invention by member Connor Greene, a senior whose passion for computer science led him to fix the attendance problem.

EASD Alert

In May of 2018, I began developing the Easton Area School District app with my mentor and former computer science teacher, Michael Tanczos. The app went live in November of 2018, and serves as a hub for the faculty and families of the Easton Area School District. We wrote the app together in C# with the Xamarin framework, and I wrote and maintain the web back-end and API which was written in C# with ASP.NET Core.

Check out our landing page at alert.eastonsd.org, or read the Lehigh Valley Live and Morning Call article about the project.

EZ Attendance

I designed and built a digital attendance system to solve my school's National Honor Society attendance problem. The site was built using PHP, MySQL, Javascript, and it used web sockets to generate single-use QR codes which members could scan and use to sign in. The system cut our attendance time in half, and made it easier to track member standing. You can read the Morning Call article about the project, our view the source code on GitHub.

Web Socket Chess

This is a real-time chess client that runs in the browser and maintains a secured socket connection with the server. All of the game information such as the timers, move history, and the state of the board are stored in a custom SQL database. This was written in Javascript and runs in NodeJS. You can see this project on my GitHub.

Locked Safe

This project involved creating a Robocode bot that uses a neural network, a form of machine learning, as its targeting system for a science fair project. Throughout the match, the robot gethers information on the enemy's movement and runs it through the neural network to become more accurate. You can see the project on my GitHub.
